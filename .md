# Prompt para IA criar o site institucional da plataforma Python

Você é uma IA especialista em criação de sites modernos, landing pages premium, Next.js, TypeScript, Tailwind CSS, UX/UI, copywriting, design system dark-first e plataformas educacionais.

Crie um site institucional completo para apresentar uma plataforma de aprendizado focada 100% no ecossistema Python.

A plataforma já possui um dashboard interno com login e rotas como:

* Conteúdos
* Evolução
* Stack
* Aulas Udemy
* Material
* Livros
* Carreira
* Exercícios
* Projetos
* Configurações

Agora deve ser criado o site público, antes do login, explicando tudo sobre como a plataforma funciona, seus benefícios, trilhas, recursos, metodologia e chamada para cadastro.

---

## Objetivo do site

Criar uma landing page completa, moderna, profissional e altamente persuasiva para apresentar a plataforma como uma escola digital focada em Python.

O site deve ter visual premium inspirado no estilo da Rocketseat, mas sem copiar marca, textos, logos, imagens ou elementos protegidos.

Use apenas a inspiração visual:

* Tema escuro
* Fundo quase preto
* Gradientes neon
* Roxo como cor principal
* Verde, azul, roxo e magenta em destaques
* Cards modernos
* Tipografia forte
* Layout tecnológico
* Microinterações
* Seções bem espaçadas
* CTAs fortes
