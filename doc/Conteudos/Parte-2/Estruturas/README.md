# Estruturas de Dados e Algoritmos com Python

Trilha completa e progressiva para dominar estruturas de dados, algoritmos e fundamentos práticos de Python. O conteúdo começa com primeiro programa, tipos, operadores e execução no terminal, avança por coleções, funções, iteradores, módulos, estruturas clássicas, busca, ordenação, recursão, grafos e programação dinâmica.

## Como Estudar

1. Execute os exemplos pequenos.
2. Reescreva as soluções com entradas diferentes.
3. Analise complexidade de tempo e espaço.
4. Transforme exemplos em funções testáveis.
5. Implemente versões ingênuas antes das otimizadas.
6. Compare estruturas de dados por custo e clareza.
7. Resolva os exercícios progressivos de cada arquivo.

---

## Competências Esperadas

Ao finalizar esta trilha, você deve saber:

- escrever e executar programas Python pelo terminal;
- dominar tipos básicos, operadores e controle de fluxo;
- escolher entre listas, tuplas, dicionários e sets;
- usar comprehensions, generators e iteradores;
- projetar funções com argumentos posicionais, nomeados, `*args` e `**kwargs`;
- organizar código com módulos e pacotes;
- explicar profundamente `if __name__ == "__main__"`;
- implementar pilhas, filas, árvores, grafos e hash tables;
- aplicar busca binária, ordenação, recursão e programação dinâmica;
- analisar Big O e trade-offs de implementação.
