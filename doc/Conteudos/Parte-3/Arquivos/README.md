Trilha completa e profissional para manipulação de arquivos em Python, cobrindo leitura/escrita, TXT, Markdown, CSV, JSON, XML, YAML, TOML, Pickle, Excel, Parquet, Avro, ORC, PDF, compressão, upload/download, streams, imagens e vídeos.

## Competências Esperadas

Ao terminar esta trilha, você deve saber:

- usar `pathlib` profissionalmente;
- ler e escrever arquivos com segurança;
- manipular formatos estruturados;
- trabalhar com Markdown, TOML e formatos colunares;
- processar arquivos grandes com streams;
- trabalhar com Excel, PDF, imagens e vídeos;
- compactar e extrair arquivos;
- fazer upload/download com controle de erro;
- evitar falhas de encoding, path traversal e arquivos maliciosos;
- criar pipelines robustos de processamento de arquivos.
