## Competências Esperadas

Ao finalizar a trilha, você deve saber:

- estruturar projetos de dados;
- manipular arrays com NumPy;
- usar SciPy para estatística, otimização e computação científica;
- limpar, transformar e analisar dados com Pandas;
- usar Polars para performance e lazy execution;
- aplicar estatística descritiva e inferencial;
- criar visualizações com Matplotlib, Seaborn, Plotly e Bokeh;
- construir dashboards;
- criar pipelines ETL/ELT;
- validar qualidade de dados;
- trabalhar com arquivos, bancos, APIs e formatos colunares;
- aplicar boas práticas de engenharia de dados;
- comunicar resultados com clareza;
- evoluir para especialista em ciência, análise e engenharia de dados.
