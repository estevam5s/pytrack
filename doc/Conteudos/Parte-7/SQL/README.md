## Sequência Recomendada

1. Comece por fundamentos SQL para entender tabelas, tipos, constraints e CRUD.
2. Avance para consultas avançadas para dominar JOINs, agregações, subqueries e índices.
3. Configure MySQL com Docker para ter ambiente reproduzível.
4. Conecte Python com MySQL usando driver direto e SQLAlchemy.
5. Estruture CRUD com repositórios, transações e queries parametrizadas.
6. Integre Excel quando precisar importar, exportar ou gerar relatórios.
7. Finalize com arquitetura profissional e projetos completos.

---

## Competências Esperadas

Ao concluir esta trilha, você deve saber:

- criar bancos, tabelas, constraints, índices e relacionamentos;
- escrever consultas SQL com filtros, ordenação, agregação, JOINs e subqueries;
- entender diferença entre DDL, DML, DQL, DCL e TCL;
- configurar MySQL com Docker e dados de inicialização;
- acessar MySQL com Python de forma segura;
- evitar SQL Injection com queries parametrizadas;
- usar SQLAlchemy Core e ORM quando fizer sentido;
- implementar CRUD profissional com transações;
- importar e exportar dados entre MySQL e Excel;
- separar configuração, domínio, repositórios e interface;
- criar projetos práticos com estrutura sustentável;
- aplicar boas práticas de segurança, performance e manutenção.
