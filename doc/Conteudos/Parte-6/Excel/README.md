# Excel Profissional com Python e Excel Solo

Trilha completa e progressiva sobre Excel solo e Excel integrado com Python: fundamentos, fórmulas, funções, tabelas, validação, análise de dados, gráficos, dashboards, Power Query, automação, relatórios, pandas, openpyxl, XlsxWriter e projetos profissionais.

O objetivo é sair do uso básico de planilhas até fluxos profissionais: modelagem limpa, fórmulas robustas, bases organizadas, dashboards executivos, importação/exportação com Python, geração automática de relatórios, validação de dados e automações confiáveis.

## Competências Esperadas

Ao concluir esta trilha, você deve saber:

- estruturar planilhas profissionais;
- separar base de dados, cálculo, parâmetros e apresentação;
- usar fórmulas essenciais e avançadas;
- aplicar validação, formatação condicional e proteção;
- criar tabelas dinâmicas e análises exploratórias;
- construir dashboards claros e úteis;
- limpar e transformar dados com Power Query;
- ler e escrever Excel com Python;
- gerar relatórios automáticos com pandas, openpyxl e XlsxWriter;
- integrar Excel com CSV, bancos, APIs e sistemas internos;
- criar automações reproduzíveis e auditáveis;
- evitar planilhas frágeis, manuais e difíceis de manter.

## Arquitetura Recomendada de Planilha

```text
arquivo.xlsx
├── README
├── Parametros
├── Base_Dados
├── Tratamento
├── Calculos
├── Tabelas_Dinamicas
└── Dashboard
```

Regra importante: não misture entrada manual, cálculo e apresentação no mesmo lugar quando o arquivo precisa ser mantido.

## Regra Principal

Excel profissional não é apenas saber fórmulas. É construir planilhas corretas, auditáveis, atualizáveis, fáceis de entender e integráveis com processos reais.
